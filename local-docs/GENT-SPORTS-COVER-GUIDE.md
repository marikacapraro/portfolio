## Step-by-Step in Figma

### 1. Create Canvas
- New file: **1200 Ã— 750px**
- Background: Light gray `#F8F9FA` (matches portfolio)

### 2. Add Desktop Calendar
**Left 60% of canvas:**
- Import: `04-calendar-grid.png`
- Position: Left side, centered vertically
- Add subtle shadow:
  - X: 0, Y: 20, Blur: 40
  - Color: Black 10% opacity
- Add border radius: 8px

### 3. Add Mobile Screens
**Right 40% of canvas:**

Import these 3 screens:
- `08-dashboard-mobile.png`
- `09-booking-mobile.png`
- `10-cart-mobile.png`

Arrange:
- Staggered/overlapping slightly
- Or side-by-side in row
- Scale to fit (keep aspect ratio)
- Add same shadow as desktop

### 4. Add Text Overlay (Optional)
- Small badge at top: "Mobile-First Redesign"
- Font: Manrope, 14px
- Color: #475569
- Background: White with slight transparency

### 5. Export
- Select all
- Export: JPG, 90% quality
- Name: `gent-sports-cover.jpg`
- Check file size < 500KB
- Save to: `/Users/marikacapraro/repos/portfolio/assets/images/projects/`
