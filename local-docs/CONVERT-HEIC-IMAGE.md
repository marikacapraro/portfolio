# Convert IMG_3418.heic to JPG

## Quick Guide

Your IMG_3418.heic file needs to be converted to JPG to use in the portfolio.

---

## Option 1: Using Preview (Mac - Easiest!)

1. **Open the image:**
   - Navigate to: `assets/images/personal picture/`
   - Double-click `IMG_3418.heic` (opens in Preview)

2. **Export as JPG:**
   - File ‚Üí Export...
   - Format: **JPEG**
   - Quality: **Best**
   - Name: `hero-split.jpg`
   - Save to: `/Users/marikacapraro/repos/portfolio/assets/images/`

3. **Done!** Refresh your browser to see the new image

---

## Option 2: Using Photos App (Mac)

1. **Open Photos app**
2. **Find IMG_3418**
3. **File ‚Üí Export ‚Üí Export 1 Photo**
4. **Settings:**
   - Photo Kind: JPEG
   - Quality: Maximum
   - Save to: Desktop
5. **Move to portfolio:**
   - Rename to: `hero-split.jpg`
   - Move to: `/Users/marikacapraro/repos/portfolio/assets/images/`

---

## Option 3: Online Converter (Any Device)

1. **Go to:** https://heictojpg.com/ (or similar)
2. **Upload** IMG_3418.heic
3. **Download** converted JPG
4. **Rename to:** `hero-split.jpg`
5. **Save to:** `assets/images/` folder

---

## After Conversion

1. **Replace the image:**
   - Save as `hero-split.jpg` in `assets/images/`
   - Overwrites current watercolor image

2. **Refresh browser:**
   - The CSS split effect will automatically apply!
   - See Designer | Developer split

3. **Adjust if needed:**
   - Image should be portrait orientation
   - Face centered
   - Good lighting/quality
