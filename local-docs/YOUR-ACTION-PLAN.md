## üé® Image Creation Tips

### For Hero Split-Face:

**EASY METHOD:**
1. Open Figma
2. Import forest photo (plaid coat)
3. Duplicate layer twice
4. Mask left half
5. Apply warm color overlay (amber/pink)

7. Mask right half
8. Add code text boxes arranged in face shape
9. Export 1000√ó1200px JPG

## üêõ Troubleshooting

### "Images aren't showing"
- Check file names (case-sensitive!)
- Verify files in correct folder
- Clear browser cache (Cmd + Shift + R)

### "Layout looks weird"
- Try different browser
- Check if CSS file loaded (View Source)
- Refresh page

### "Mobile menu won't open"
- Check if JavaScript loaded
- Open DevTools Console (F12) for errors
- Try different browser

### "I want to change colors"
- Edit `styles/main.css`
- Change values in `:root` section (top of file)
- Colors defined once, used everywhere
